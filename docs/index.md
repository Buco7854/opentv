::: warning OpenTV is a player, not a provider
OpenTV does not provide any channels, streams or subscriptions, and none are
included. You bring your own M3U playlist or Xtream login from a provider you
already use.
:::

## Two download channels

OpenTV is distributed as an APK you install yourself. It is not on the Play Store.

- **Latest release** is the most recent tagged, stable version. Use this for everyday viewing.
- **Dev channel** is rebuilt from the latest commit on the main branch, so it has the newest changes first.

Both builds are signed with the same key, so you can switch between them and install updates in place without losing your playlists or settings. See [Updating the app](/guide/updating) for details.
