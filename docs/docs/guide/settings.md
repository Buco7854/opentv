# Settings

Open Settings from the home screen to tune playback, downloads and the network behavior.

## Tracks

Set your preferred audio and subtitle languages. When a stream offers a matching track, OpenTV selects it automatically.

## Playback

- **Seek step** for the double-tap skip.
- **Buffering** preset, from fast start to stable. Stable buffers more to ride out shaky connections.
- **Default video scaling**, between Fit, Zoom and Stretch.
- **Software decoder fallback**, which retries with a software decoder if the hardware one fails.

## Downloads

- **Simultaneous downloads**, either automatic or a fixed number. Automatic uses your provider's connection limit and keeps one slot free for watching.
- **Location**, where you choose a folder for downloaded files.
- **Move existing downloads here**, which relocates already downloaded files into the current folder.

## Network

- **User-Agent** sent with every request. Some providers reject unknown apps with a 404 or 403, so you can match what your provider expects. Presets are included for VLC, IPTV Smarters, Kodi and TiviMate, plus a free text field. Leave it blank to use the default.

## Subtitles

Adjust subtitle size, background style and bold, with a live preview.
