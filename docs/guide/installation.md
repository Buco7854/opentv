# Installation

OpenTV is distributed as an APK. It is not on the Play Store, so you install it directly. The process takes about a minute.

## 1. Download the APK

On your Android phone, tablet or TV, get the build you want:

- [Latest release](https://github.com/Buco7854/opentv/releases/latest/download/app-debug.apk) for the stable version
- [Dev channel](https://github.com/Buco7854/opentv/releases/download/dev/app-debug.apk) for the latest commit

See [Download](/download) if you are unsure which to pick.

## 2. Allow installs from this source

Android blocks app installs from outside the Play Store until you allow them. When you open the APK, Android will offer to take you to the right setting. You can also set it ahead of time:

- Open **Settings**, then **Apps**, then **Special app access**, then **Install unknown apps**.
- Choose the app you downloaded with, such as your browser or file manager, and turn on **Allow from this source**.

This permission is per app and you only grant it once.

## 3. Open the APK and install

Tap the downloaded file and confirm the install. You may see a warning that the app was not scanned by the Play Store. This is normal for sideloaded apps. Choose **Install anyway** or **Install without scanning**.

## Installing on Android TV

Android TV does not have a browser, so use a sideload helper:

1. Install a file manager such as X-plore or a tool such as Downloader from the Play Store on the TV.
2. Open one of the download links above inside that app.
3. Open the downloaded APK and install it.

OpenTV adds a leanback launcher entry and supports D-pad navigation throughout.

## After installing

Launch OpenTV and continue with [Add a playlist](/guide/playlists).
